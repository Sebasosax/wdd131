Doña Betty Empanadas — Website Plan

Author: Juan Diego Sebastián Sosa

Website Name

Name: Doña Betty Empanadas

Description: A family-owned restaurant specializing in homemade empanadas and traditional dishes.

Website Purpose

This website will promote the family business, present the menu and special offers, provide contact and ordering information, and share the story behind Doña Betty Empanadas.

User Scenarios

  • A local visitor wants to see the menu and prices to decide what to order.
  • A customer is looking for contact details and location to place an order or visit the restaurant.

Color Scheme

  • Primary: Warm Red — #C0392B (used for headings and main buttons)
  • Secondary: Cream — #F8EDEB (backgrounds and panels)
  • Accent: Dark Brown — #4E3B31 (text, footers, small accents)

Typography

  • Roboto — used for body text
  • Roboto Bold — used for headings and emphasized text

Site Structure

  • Home (About) — Main introduction and hero image
  • Menu & Specials — Menu items, pricing, and current promotions
  • Contact — Address, phone number, and order form

Wireframes

CSS / Styling Notes

The design will use the selected color palette and Roboto font. A simple responsive layout will be implemented for mobile and desktop screens.

Testing & Validation

  1. Validate HTML using the W3C validator.
  2. Check accessibility using Lighthouse or Axe.
  3. Verify color contrast for readability.
